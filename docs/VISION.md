# Vision

## Mission

Thermal Brain provides a vendor-independent thermal intelligence layer for buildings.

Instead of optimizing a heating system directly, Thermal Brain first learns how a building behaves and then derives explainable recommendations.

## Principles

- Building first
- Local first
- Explainable recommendations
- Physics before AI
- Vendor independent

## Development Stages

1. Observe
2. Learn
3. Predict
4. Recommend
5. Optimize

## Non Goals (v0.x)

- Automatic heating control
- Cloud dependency
- Black-box machine learning
